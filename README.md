# Baby Wish List #

A web app to build baby wish lists.

This project uses:
 
*	[Node.js](http://nodejs.org/)
	*  [Express](http://senchalabs.github.com/connect/)
	*  [Connect](http://expressjs.com/)
	*  [Jade](http://jade-lang.com/)
*	[MongoDB](http://www.mongodb.org/)
* 	[JQuery](http://jquery.com/)
